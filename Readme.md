### ASP.NET ile Web Programlama Eğitimi 

### Eğitim Tanıtımı

ASP.NET Core ile Web Programlama eğitimi, modern ve ölçeklenebilir web uygulamalarının geliştirilmesini hedefleyen bütüncül bir öğrenme deneyimi sunar. Eğitim süreci, web temelleri ile başlamakta ve katmanlı mimari, MVC, Razor Pages ve Web API gibi ASP.NET Core ekosisteminin temel bileşenleri üzerine sistematik biçimde inşa edilmektedir. Katılımcılar, .NET çalışma modeli ve middleware kavramlarını kavrarken aynı zamanda Entity Framework Core aracılığıyla veritabanı işlemlerini güvenli ve sürdürülebilir şekilde gerçekleştirmeyi öğrenir. İçerik, teorik bilginin uygulama ile pekiştirilmesini esas alır ve her aşamada geliştirilen örnekler gerçek dünya senaryolarını temel alır. Bu yaklaşım, öğrenilen kavramların soyut kalmasını engeller ve kalıcı bir teknik yetkinlik oluşturur.

Eğitimin ilerleyen aşamalarında RESTful servis geliştirme, API tüketimi, kimlik doğrulama ve rol tabanlı yetkilendirme gibi kurumsal uygulamalarda kritik öneme sahip konular ele alınır. Süreç, mini proje geliştirme ve dağıtım adımları ile tamamlanarak katılımcıların uçtan uca bir web projesini planlama, geliştirme, test etme ve yayına alma becerilerini kazanması hedeflenir. Kod kalitesi, mimari tutarlılık, güvenlik ve bakım kolaylığı gibi profesyonel yazılım geliştirme kriterleri eğitim boyunca vurgulanır. Bu depo, eğitim kapsamında üretilen tüm örnekleri, uygulama yapılarını ve rehber niteliğindeki içerikleri bir araya getirerek öğrenme sürecini destekleyen kalıcı bir başvuru kaynağı olarak tasarlanmıştır. 

### Eğitim İçeriği

1. **Web Temelleri**
   Bu bölümde HTML5, CSS3 ve temel JavaScript kavramları tanıtılarak modern web sayfalarının yapısal ve görsel bileşenleri açıklanır.

2. **.NET ve ASP.NET Core’a Giriş**
   .NET Core çalışma modeli, middleware kavramı ve ilk ASP.NET Core uygulamasının oluşturulması ele alınır.

3. **MVC Temelleri**
   Model, View ve Controller bileşenlerinin sorumlulukları açıklanarak MVC mimarisi üzerinden katmanlı uygulama geliştirme yaklaşımı incelenir.

4. **Razor Pages**
   Razor sözdizimi, layout ve partial view yapıları kullanılarak dinamik ve bakımı kolay sayfalar geliştirilir.

5. **Veritabanı Temelleri**
   İlişkisel veritabanı kavramları ve SQL temelleri üzerinden Entity Framework Core’a giriş yapılır.

6. **EF Core ile CRUD İşlemleri**
   DbContext, migration ve CRUD operasyonları kullanılarak veritabanı destekli uygulamalar geliştirilir.

7. **Model Binding ve Validation**
   Kullanıcıdan alınan verilerin modele bağlanması ve DataAnnotations ile doğrulanması süreçleri açıklanır.

8. **Web API’ye Giriş**
   REST mimarisi, HTTP kavramları ve API Controller yapıları kullanılarak temel Web API geliştirme ele alınır.

9. **API Tüketimi**
   MVC ve Razor Pages uygulamalarından HttpClient aracılığıyla Web API servislerinin nasıl tüketileceği gösterilir.

10. **Kimlik Doğrulama Temelleri**
    Cookie tabanlı kimlik doğrulama mantığı kullanılarak basit bir kullanıcı giriş sistemi oluşturulur.

11. **Güvenlik ve Yetkilendirme**
    Rol tabanlı yetkilendirme yaklaşımı ile uygulama içinde erişim kontrolü mekanizmaları uygulanır.

12. **Basit Frontend Entegrasyonu**
    Fetch API ve AJAX kullanılarak istemci tarafında API’den veri alma ve işleme süreçleri incelenir.

13. **Proje Geliştirme I**
    Mini bir web projesinin planlanması ve temel mimari iskeletinin oluşturulması gerçekleştirilir.

14. **Proje Geliştirme II**
    Projenin CRUD, kimlik doğrulama ve API entegrasyonu tamamlanarak yayınlama süreçleri ele alınır.

15. **Proje Sunumu ve Değerlendirme**
    Geliştirilen projenin sunumu yapılır ve doğruluk, güvenlik, kullanılabilirlik ve kod kalitesi açısından değerlendirilir.

### Eğitimin Niteliği
Bu eğitim, kuramsal temelleri uygulama odaklı bir yaklaşımla bütünleştiren, modern yazılım geliştirme standartlarını esas alan nitelikli bir programdır. Katılımcılara ASP.NET Core ekosistemi içinde güvenilir, sürdürülebilir ve ölçeklenebilir web uygulamaları geliştirme yetkinliği kazandırmayı amaçlar. Eğitim süreci, mimari tutarlılık, kod kalitesi ve güvenlik ilkelerini merkeze alır.
